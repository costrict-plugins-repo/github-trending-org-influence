# /build-narrative
Build a narrative that moves a senior audience.
## Steps
1. **Map the room** — Identify deciders and influencers using `influence-map` skill.
2. **Craft the story** — Build the audience-first narrative using `executive-narrative` skill.
3. **Plan the spread** — Decide how to seed it using `design-advocacy` skill.
## Output
A narrative pack: an influence map, a tailored executive narrative, and a plan to seed it before the decision.
Consider following up with `/secure-resources` if the narrative supports a funding ask.
