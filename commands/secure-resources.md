# /secure-resources
Prepare a resourcing ask that wins.
## Steps
1. **Build the ask** — Frame the gap, options, and trade-offs using `securing-resources` skill.
2. **Line up support** — Map and win sponsors using `influence-map` skill.
3. **Manage up** — Brief your own leadership to advocate using `managing-up` skill.
## Output
A resourcing case: a quantified gap, tiered options, a sponsor plan, and an upward brief.
Consider following up with `/build-narrative` to broaden support beyond the budget forum.
